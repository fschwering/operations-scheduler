# Operations Scheduler

A flexible, web-based scheduler for managing operations, groups, and dependencies.

## Features
- **Interactive Gantt Chart**: Drag and resize tasks directly on the chart.
- **Dependency Tracking**: Tasks can depend on each other with custom gaps.
- **Undo/Redo System**: Full state-based undo/redo support.
- **Export Options**: Export your schedule as a PNG image or CSV data.
- **Customizable**: Configure chart title, headers, grid lines, and row styles.
- **Local Storage**: Save and load your schedules as JSON files.

## Files
- `index.html`: Main application entry point.
- `css/schedule.css`: Styling for the application and chart.
- `js/`: Core application logic (model, view, events, utilities).
- `html2canvas.min.js`: External library for PNG export.

## How to Use
Simply open `index.html` in a modern web browser. No server or installation required.
